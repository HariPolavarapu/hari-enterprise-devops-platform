# Frontend Angular

Angular based frontend application for the Hari Enterprise DevOps Platform.

## Stack

- Angular
- TypeScript
- NGINX
- Docker
- Jenkins
- Kubernetes
- AWS EKS
- ArgoCD

---

## Features

- Dashboard UI
- Employee Management UI
- Responsive Design
- Dockerized Deployment
- CI/CD Ready

---

## Build

npm install

npm run build

---

## Run

ng serve

---

## Docker

docker build -t frontend-angular .

---

## Deployment Flow

GitHub → Jenkins → Docker → AWS ECR → ArgoCD → AWS EKS
